# Introduction

Docsflare renders Mintlify-style MDX documentation as a standard Fetch API application. Keep your `docs.json` or `mint.json`, move your `.mdx` files over, and deploy to Cloudflare Workers, Node.js, or another Fetch-compatible runtime.

## What is included

- Static MDX rendering at build time
- Mintlify-style navigation parsing
- A `docsflare` CLI for local dev, builds, deploys, and project checks
- Portable routing for pages, sitemap, and `llms.txt`
- Built-in local search with optional provider integrations
- Cloudflare AI Search support through an optional binding
