# Quickstart

Install dependencies and start the Worker locally:

```bash
npm install
npm run dev
```

The content build reads `docs.json` or `mint.json` and all referenced Markdown/MDX pages from the project root.

## Move a Mintlify project

Copy your existing Mintlify files into this repository:

- `docs.json` or `mint.json`
- all `.mdx` and `.md` pages
- images and other public assets you reference from your docs

Most projects only need to adjust unsupported custom React imports. Common Mintlify documentation components are mapped to built-in HTML equivalents during the build.

## Deploy

Create an AI Search instance named `docsflare-docs`, then deploy:

```bash
npm run deploy
```
